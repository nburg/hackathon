Test the P3 DOM & NLP extraction module on any website!

npx http-server . -p 8080 --cors

📚 P3 Extract Words

Method 2: Browser Console

Step 1: Open any website
Step 2: Open DevTools (F12) → Console
Step 3: Paste this code:
import('http://localhost:8080/dist/index.js').then(P3 => { window.P3 = P3; const candidates = P3.extractCandidates(document); console.log(`Found ${candidates.length} candidates`); console.table(candidates.slice(0, 20)); });

Method 3: Local Test Page

Open in your browser:

http://localhost:8080/browser-test.html

📊 What You'll See

After extraction, check the console for:

  • Total candidates: Number of words extracted
  • POS distribution: How many nouns, verbs, adjectives, etc.
  • Sample words: First 20 candidates in a table
  • Proper nouns: Names and places flagged

✅ Expected Behavior

  • ✅ Extracts words from paragraphs, headings, lists
  • ✅ Skips navigation links
  • ✅ Ignores code blocks (<code>, <pre>)
  • ✅ Identifies parts of speech (Noun, Verb, etc.)
  • ✅ Flags proper nouns (names, places)
  • ✅ Provides DOM node references for replacement

🧪 Example Commands

Once loaded, try these in the console:

// Get all nouns const nouns = P3.filterCandidates( P3.extractCandidates(document), c => c.pos === 'Noun' ); console.log('Nouns:', nouns.map(n => n.word)); // Exclude proper nouns const common = P3.filterCandidates( P3.extractCandidates(document), c => !c.isProperNoun ); // Group by element const groups = P3.groupCandidatesByElement( P3.extractCandidates(document) ); console.log(`Distributed across ${groups.size} elements`);

🔧 Troubleshooting

CORS Error?

Make sure the server is running with --cors flag:

npx http-server . -p 8080 --cors

Module not found?

Build the project first:

npm run build

No candidates extracted?

The page might not have readable content. Try a news article or Wikipedia page.
