One-time setup — takes about 2 minutes

You're all set!

Chrome's on-device translator is ready. Navigate to any English webpage and the extension will start replacing words with Spanish.

1
Chrome 131 or later
The built-in Translation API requires Chrome 131+. Check or update at chrome://settings/help.
Checking…
2
Enable two Chrome flags
Open each URL in a new tab, set the value shown, then click Relaunch.
chrome://flags/#translation-api
→ Set to Enabled
chrome://flags/#optimization-guide-on-device-model
→ Set to Enabled BypassPerfRequirement
Checking…
3
Download the language model
Chrome needs to download the English → Spanish model (~50 MB). Open the internals page to trigger and monitor the download:
chrome://on-device-translation-internals
Find en → es and click Install if it isn't downloading already.
